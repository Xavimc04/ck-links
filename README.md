## Links website

Custom and simple website made for a customer as Freelancer. 

## Features

- Display social media links
- Translations
- Responsive design

## Stack

- Astro
- Tailwindcss

## Installation 

Clone repository 

```
git clone https://github.com/Xavimc04/ck-links.git
```

Install dependencies

```
npm install
```

Run development server

```
astro dev
```

or 

```
npm run dev
```

Enjoy it!
